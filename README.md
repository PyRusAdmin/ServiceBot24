### Бот для продаж услуг, обратной связи, помощи в работе.

## Структура проекта

```
ServiceBot24/
├── db/                    # Работа с базой данных
├── handlers/             # Обработчики бота
├── keyboards/            # Клавиатуры бота
├── messages/             # Сообщения бота
├── setting/              # Настройки и конфигурации
├── states/               # Состояния FSM
├── system/               # Системные файлы и диспетчер
├── main.py               # Главный файл запуска бота
├── requirements.txt      # Зависимости проекта
├── setup.py              # Настройка окружения
├── README.md             # Документация проекта
├── Запуск.bat            # Скрипт запуска бота
└── Установка.bat         # Скрипт установки зависимостей
```

Основные команды:

* /start - запуск бота для обычного пользователя

В проекте используются платежные системы:

* yookassa (https://yookassa.ru/)
* cryptomus (https://cryptomus.com)
